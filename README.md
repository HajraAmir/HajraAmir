<h4>🧩 Notescape — AI-Powered Learning Workspace (Team FYP)</h4>

<em>A full-stack study platform that turns a student's own notes into an interactive, AI-driven learning system.</em>

- My role: built the **RAG-based chatbot** (LLaMA 3 70B via Groq API) for document-grounded Q&A, implemented the
  **spaced-repetition scheduling engine** for personalized study sessions, and led **handwritten text recognition
  (HTR) research** across the IAM dataset and other benchmarks to improve CER/WER through training strategies,
  data augmentation, and decoding/error analysis.
- Tech: `React` `TypeScript` `FastAPI` `PostgreSQL + pgvector` `Groq API`
- Repo: [NotescapeAI/Notescape](https://github.com/NotescapeAi/Notescape)

<h4>🤖 PersonaBot — AI-Powered Professional Profile Intelligence Platform (RAG)</h4>

<em>An end-to-end RAG application that turns resumes and LinkedIn profiles into a queryable knowledge base.</em>

- Ingests resumes (PDF/DOCX) and LinkedIn profile HTML, then extracts structured career data — skills,
  experience, and education.
- Semantic chunking + vector similarity search with **PostgreSQL and pgvector** for context-aware retrieval.
- Conversational, **RAG-based question answering** with persistent chat memory per profile/user.
- Profile comparison across skills, experience, and seniority; secure JWT authentication.
- Fully **Dockerized** full stack for reproducible deployment.
- Tech: `FastAPI` `PostgreSQL + pgvector` `Groq (LLaMA-3.3-70B)` `React` `Vite` `TypeScript` `Docker`
- Repo: [HajraAmir/PersonaBot](https://github.com/HajraAmir/PersonaBot)

<h4>🧠 SmartResumeCheck — NLP + AI Resume Matching System</h4>

<em>AI-powered resume–job description matching system using semantic NLP.</em>

- Built a full-stack system using **React + Flask**.
- Implemented a **spaCy-based NLP pipeline** for job requirement extraction.
- Used **SBERT embeddings** for semantic similarity matching, scored via cosine similarity.
- Supports **Student Mode & HR Mode**, with batch processing for multiple resumes (CSV / folder upload).
- Generates **match percentage + improvement suggestions**.
- Tech: `React` `Flask` `spaCy` `SBERT` `NumPy`
